implementation of a Mediametrie JavaScript beacon on a Youtube player.

This is a Youtube player. By using the Youtube API, developers are able to implement a Mediametrie JavaScript beacon to measure the stream consumption. The Youtube player API uses callbacks in which methods of the Mediametrie beacon are called to send playing events and positions to the Mediametrie servers.

By opening the network panel of the developer console (F12) developers will see requests sended to mediametrie servers (*.estat.com) when stream is played, paused, seeked and stoped. We also trigger polling requests to regularly send the playhead position to our servers
